# Async Posts Filter (API)

A small app that fetches posts and filters them by title.

## Features

- Fetches data from JSONPlaceholder API
- Live search by title
- Debounced input (reduces unnecessary requests)
- Renders posts list

## Notes

Uses an external demo API: https://jsonplaceholder.typicode.com  
Requires internet connection.

## How to run

This project uses ES Modules.

- Works on GitHub Pages
- Locally: run a simple server (recommended)

Example:

```bash
npx serve
```
