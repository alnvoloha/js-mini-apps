# Password Generator (Educational)

Password generator + simple local vault storage (educational project).

## Features
- Generate strong passwords (length + character sets)
- Save / retrieve passwords using localStorage
- Import / export vault

## Notes
This project is **for learning purposes** and should not be used to store real sensitive passwords.

## Tech
HTML • JavaScript • Tailwind (CDN)

## Run locally
Open `index.html` in your browser.
