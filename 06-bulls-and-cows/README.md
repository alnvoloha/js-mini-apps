# Bulls & Cows (4-digit)

A small logic game: guess a random 4-digit number with unique digits.

## Features

- Generates a secret 4-digit number (no repeated digits)
- Input via 4 digit fields
- Calculates bulls (correct digit & position) and cows (correct digit, wrong position)
- Shows attempt history

## How to run

Open `index.html` in a browser.
