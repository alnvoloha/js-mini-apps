# Frogs in the Swamp

Classic logic puzzle: swap frogs from left to right using valid moves.

## Rules
- A frog can move to an adjacent empty stone
- Or jump over one frog into an empty stone

## Features
- Move validation
- Reset / Try again
- Win detection

## Tech
HTML • JavaScript • Tailwind (CDN)

## Run locally
Open `index.html` in your browser.
