# Users → Albums → Photos (Hash Router)

A mini SPA-like browser using hash-based navigation and public API endpoints.

## Features

- Loads users list
- Opens user albums
- Opens album photos
- Back navigation (users / albums)
- Hash-style links (simple routing pattern)

## Notes

Uses JSONPlaceholder:

- /users
- /albums?userId=...
- /photos?albumId=...

Requires internet connection.

## How to run

This project uses ES Modules.

- Works on GitHub Pages
- Locally: run a simple server

Example:

```bash
npx serve
```
