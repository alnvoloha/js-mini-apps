# Tic-Tac-Toe (Variable Board)

Classic tic-tac-toe with a customizable board size.

## Features

- Choose board size (3×3 up to 10×10)
- Alternating turns (X / O)
- Win detection (rows, columns, diagonals)
- Draw detection
- Restart button after game ends

## How to run

Open `index.html` in a browser.
